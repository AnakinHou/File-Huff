# Picture-Music

一个可以将所有文件转换为图片并经过哈夫曼压缩，界面采用了拟态元素的小项目，程序运行中的光影变换让人感到十分惬意

这是一个基于maven框架，javafx图形界面的小玩意，我的运行环境是win10，jdk13，项目到本地后记得修改代码中的路径以免报错

我尽了自己最大的努力让代码变得漂亮同时优化数据结构，以下是我的测试表：
